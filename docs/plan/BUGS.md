# Bug Tracker

Issues and bugs found during development. Use `/bug log` to add, `/bug list` to browse.

---

## Severity Levels

- **S0 – Critical:** System broken, data loss risk, security vulnerability. Fix immediately.
- **S1 – High:** Major functionality broken, workaround difficult. Fix before next release.
- **S2 – Medium:** Minor functionality broken or degraded, workaround available. Fix in current sprint.
- **S3 – Low:** Cosmetic, missing nice-to-have, or very minor impact. Fix when time permits.
